# コミットメッセージを日本語化

コミットメッセージを英語から日本語に翻訳します。

## 実行内容

1. ユーザーが提供したコミットメッセージ（英語）を日本語に翻訳
2. 技術的な用語（SAML、IDP、Entra IDなど）は適切に保持
3. コミットメッセージの形式（タイトルと箇条書き）を維持
4. 自然な日本語として読みやすい形で翻訳

## 使用例

英語のコミットメッセージを提供すると、以下のような形式で日本語化されます：

```
SAML設定とドキュメントの更新

- docker-compose.ymlからプレースホルダーのSAML IDPメタデータURIを削除し、より柔軟な設定を可能にしました。
- README.mdを更新し、新しいSAML IDPメタデータURIの設定を反映し、有効なEntra IDメタデータURLの必要性を強調しました。
- SAML設定に不要になったため、idp-placeholder.xmlファイルを削除しました。
```
